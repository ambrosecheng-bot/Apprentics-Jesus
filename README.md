# 與主同行——大使命門徒協作平台
### *Apprentics of Jesus · Great Commission Discipleship Platform*

🌐 **Live Site:** [https://ambrosecheng-bot.github.io/Apprentics-Jesus/](https://ambrosecheng-bot.github.io/Apprentics-Jesus/)

---

## 🗺️ 專案簡介

「與主同行」是一個面向**英國香港移民基督徒群體**的門徒培育協作平台，由修咸頓華人基督教會（SCCC）同工團隊建立與維護。

平台以**耶穌的門訓方法**為藍本，整合聖經知識庫、門訓資源及跨教會同工協作功能，服務四類使用者：初信者、受訓門徒、事工同工、教會領袖。

> *「你們去，使萬民作我的門徒」——馬太福音 28:19*

---

## ✨ 主要功能

### 📖 聖經知識庫
- **30條條目，涵蓋九站門徒進程**——從「黑暗與光明」到「儆醒與終末盼望」
- 每條含 **BDAG 標準希臘文詞彙解析**、門訓原則及今日應用
- 即時搜尋、按站篩選
- 點擊展開完整 Modal 詳覽

### 🗂️ 九站門訓進程

| 站次 | 名稱 | 主題 |
|------|------|------|
| 第一站 | 黑暗與光明 | 天國、悔改、永生 |
| 第二站 | 蒙召與回應 | 呼召、信心、重生 |
| 第三站 | 捨棄與跟從 | 捨己、背十字架、家庭 |
| 第四站 | 與主同行受訓 | 同在、學效、禱告、禁食、謙卑、清心 |
| 第五站 | 試煉與幽谷 | 饒恕、憐憫、受苦、跌倒與復和 |
| 第六站 | 深處的更新 | 聖靈、等候得力、忠心 |
| 第七站 | 天國的群體生命 | 愛神愛人、彼此相愛、合一、和平 |
| 第八站 | 差遣與倍增 | 差遣與回報、倍增 |
| 第九站 | 儆醒與終末盼望 | 末世警醒、審判與復活 |

### 🌏 三語支援
- 繁體中文 · 簡體中文 · English
- 介面、知識庫條目、篩選標籤全面三語化
- 語言設定以 `localStorage` 持久保存

### 🎨 視覺設計
- 加利利海邊羊皮紙古典美學（古代近東色調）
- 動態英雄區（Ken Burns 動畫）
- 全響應式設計，支援桌面及手機

## 🚀 本地運行

```bash

git clone https://github.com/ambrosecheng-bot/Apprentics-Jesus.git

# 進入目錄
cd Apprentics-Jesus

# 以任意瀏覽器開啟 index.html
open index.html   # macOS
```

無需安裝任何依賴套件，直接在瀏覽器開啟即可運行。

## 🌱 事工背景與異象

本平台由修咸頓華人基督教會（**SCCC · Southampton Chinese Christian Church**）建立，附屬於海外華人基督教使命（COCM）。

### 服務對象
- 英國香港移民基督徒（BNO 及其他移民路徑）
- 教會初信者及尋道者
- 跨教會華人同工

### 核心理念
> **「一個影響一個」**——以耶穌的倍增方式，建立可複製的門訓事工

- 工具服務葡萄樹，而非取代葡萄樹
- 關係先於任務，同在先於事工
- 簡單、可複製、跨代際傳承

### 目標
連接英國各地華人教會，共同建立以耶穌為師的門徒培育文化，使每一個香港移民基督徒都能在異鄉落戶天國，並進一步影響下一個人。
