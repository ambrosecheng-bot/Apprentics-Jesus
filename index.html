01
我只是路過，還不確定這裡是否適合我
我對信仰沒有什麼特別的感覺
+

沒關係。很多人來這裡，只是因為朋友邀請，或者生活出現了問題，或者純粹好奇。你不需要有任何立場，也不需要做任何決定。這裡歡迎每一個願意看看的人。

隨便看看 →
02
我對信仰有一些印象——好的和壞的都有，但不太確定是真是假
聽過不少，但都是別人說的
+

你的懷疑是誠實的。聖經不害怕問題——事實上，歷史上最認真的信徒，往往都是從最多懷疑開始的。這裡有資源讓你自己判斷，不是叫你盲目接受別人說的。

看看真實的問題與回答 →
03
我覺得信仰可能對我有幫助，想多了解
開始覺得這和我的生活可能有關
+

這個感覺值得跟隨。耶穌從來不是叫人「加入宗教」，而是邀請人進入一段真實的關係。如果你想了解這段關係是什麼，下一步很簡單——讓聖經自己說話。

04
我想用自己的眼睛讀聖經，看看裡面說什麼
想自己判斷，不被別人說服
+

這是最好的起點。聖經不怕被認真閱讀——幾千年來，最多讀過它的人，往往成為最深信它的人。我們有簡單的查經指引，讓你從頭開始，不需要任何背景知識。

開始讀聖經 →
05
我想認真決定：跟從耶穌，對我來說是什麼意思？
心裡有感動，但想搞清楚再決定
+

這個問題非常真實，值得認真回答。跟從耶穌不是一個宗教加入手續，而是一個生命方向的選擇。我們可以一起坐下來，慢慢談清楚這個決定意味著什麼——沒有壓力，沒有期限。

新門徒
已經信主，在生命深處扎根成長
06
我信了，但還有很多不懂，需要有人陪伴
像剛到英國一樣，什麼都要重新學
+

剛信主，感覺有很多不知道，這完全正常。耶穌的門徒也是這樣開始的——他們不是先學會再跟從，而是在跟從的過程中學習。我們有門訓小組，一個願意陪你走的人，讓你不必一個人摸索。

找一個同行者 →
07
我想讓信仰真正改變我這個人，不只是週日去教會
想要更深入，不只是表面
+

這個渴望本身，已經是聖靈在你裡面工作的記號。真正的門訓不是增加更多宗教活動，而是讓耶穌的教導從裡面改變你——思想、習慣、關係、身份。我們有系統的九站進程，陪你走這段深度的旅程。

開始深度門訓旅程 →
同行者
渴望把生命的改變帶給身邊的人
08
我想把在自己身上發生的，也帶給身邊的人
想開始陪伴別人，但不知道怎樣做
+

這正是「一個影響一個」的起點。你不需要成為神學家，也不需要有完美的生命——你只需要願意，願意陪著另一個人，走你已經走過的路。我們有門訓者訓練，幫你踏出這一步。

成為門訓者 →
09
我希望我影響的人，也能去影響別人
不只是做好事，而是讓改變可以繼續傳下去
+

這就是大使命的核心——不是一個人影響很多人，而是每個人影響下一個人，讓改變像漣漪一樣擴散出去。提摩太後書 2:2 說的正是這個。我們歡迎你加入這個跨代際的倍增事工。
